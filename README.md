# flashcard-game

A simple clickable Python flashcard game for English to Spanish vocabulary.

## Run

```bash
python main.py
```

## How it works

- One English word is shown on screen.
- Click **Reveal Translation** to show the Spanish translation and phonetic spelling.
- Click **Next Card** for another word.
- Press **Enter** as a shortcut:
	- First press reveals translation.
	- Second press moves to the next card.

Example card format:

- `always : siempre (syem-pre)`
